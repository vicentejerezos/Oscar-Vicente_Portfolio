Project IV: Credit Card Fraud Detection

Semi-supervised Learning and Classification Model. Dealt with high imbalanced dataset. Created a tool that identifies, using a Dataset provided with transactions made by credit cards in September 2013 by European cardholders, fraudulent credit card transactions. There are three Resampling Techniques: Under sample majority class. Over sample minority class. Generate synthetic samples (SMOTE). Used Under sampling and Over sampling to deal with class imbalance in this project.

Project III: Walmart - Store Sales Forecasting

Supervised Learning and Regression Model. Created a tool that predicts, using a Dataset provided with historical sales data for 45 Walmart stores located in different regions, the weekly sales of each store. Analyzed correlation between price and every and each one of the features. Dealt with skewed data. Handled negative data values while reducing high skewness. Came up with an effective solution, u = log(y + 1 - min(y)), to convert them into positive when transforming skewed data.
